Address Constructor

Parameters:

  • addressValue: An email address.

Exceptions:

  • System.ArgumentNullException: The parameter addressValue is null.

  • System.ArgumentException: The email address contains invalid syntax. For example, it doesn't contain an '@' sign or either side of the '@' contains invalid characters, the address is too long, or the address contains comments (text within parentheses).
