Specifies what kind of data a message body is.

A media type consists of a top-level type (the general category of the ata), a subtype (the specific type), and an optional list of arameters. For example, the media type text/plain; charset = utf-8 is a text media type ("text"), namely, a plain text type ("plain"), and he parameters say that the data uses UTF-8, a Unicode character ncoding ("charset = utf-8"). Other top-level types include "audio", video", and "application".

A media type is sometimes known as a "MIME type", for Multipurpose nternet Mail Extensions, the standard that introduced media types.

This type is immutable, meaning its values can't be changed once it' s reated. To create a changeable media type object, use the ediaTypeBuilder class.
