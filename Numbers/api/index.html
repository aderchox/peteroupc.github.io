API Documentation

  • com.upokecenter.numbers.EContext - Contains parameters for controlling the precision, rounding, and exponent range of arbitrary-precision numbers.

  • com.upokecenter.numbers.EDecimal - Represents an arbitrary-precision decimal floating-point number.

  • com.upokecenter.numbers.EFloat - Represents an arbitrary-precision binary floating-point number.

  • com.upokecenter.numbers.EInteger - Represents an arbitrary-precision integer.

  • com.upokecenter.numbers.ERational - Represents an arbitrary-precision rational number.

  • com.upokecenter.numbers.ERounding - Specifies the mode to use when "shortening" numbers that otherwise can't fit a given number of digits, so that the shortened number has about the same value.

  • com.upokecenter.numbers.ETrapException - Exception thrown for arithmetic trap errors.
