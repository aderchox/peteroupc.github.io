  • Ceiling
    Deprecated. If there is a fractional part, the number is rounded to the highest representable number that's closest to it.
  • Down
    Deprecated. The fractional part is discarded (the number is truncated).
  • Floor
    Deprecated. If there is a fractional part, the number is rounded to the lowest representable number that's closest to it.
  • HalfDown
    Deprecated. Rounded to the nearest number; if the fractional part is exactly half, it is discarded.
  • HalfEven
    Deprecated. Rounded to the nearest number; if the fractional part is exactly half, the number is rounded to the closest representable number that is even.
  • HalfUp
    Deprecated. Rounded to the nearest number; if the fractional part is exactly half, the number is rounded to the closest representable number away from zero.
  • Odd
    Deprecated. If there is a fractional part and the whole number part is even, the number is rounded to the closest representable odd number away from zero.
  • OddOrZeroFiveUp
    Deprecated. For binary floating point numbers, this is the same as Odd.
  • Unnecessary
    Deprecated. Indicates that rounding will not be used.
  • Up
    Deprecated. If there is a fractional part, the number is rounded to the closest representable number away from zero.
  • ZeroFiveUp
    Deprecated. If there is a fractional part and if the last digit before rounding is 0 or half the radix, the number is rounded to the closest representable number away from zero; otherwise the fractional part is discarded.
