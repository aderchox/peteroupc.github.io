public static class InvariantText

Contains string processing operations that should be consistent egardless of locale. This class, along with the CulturedText class, upports the correct use of string functions in the framework. nvariantText is intended for use in locale-independent string processing, uch as data and file system processing, while CulturedText is intended or text display.
