PeterO.CulturedText

PeterO.CulturedText

public static class CulturedText

Contains string processing operations that should be formatted according to the current locale's formatting rules.

This class, along with the InvariantText class, supports the correct use of string functions in the framework. InvariantText is intended for use in locale-independent string processing, such as data and file system processing, while CulturedText is intended for text display. Use CulturedText for formatting the messages of exceptions and within the Console.Write method.
