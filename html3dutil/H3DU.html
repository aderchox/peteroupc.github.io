Namespace: H3DU

H3DU

The Public Domain HTML 3D Library contains classes and utility methods to ease the development of HTML 3D applications, such as Web sites, in browsers that support 3D drawing using the HTML5 Canvas.

This page describes miscellaneous utility methods included in the library.

(static) getTimePosition(timer, timeInMs, intervalInMs) → {Number}

Source: h3du.js, line 393

Gets the position of a time value within an interval. This is useful for doing animation cycles lasting a certain number of seconds, such as rotating a shape in a 5-second cycle. This method may be called any number of times each frame.
Parameters:
timer ( Object )
An object that will hold two properties:
  • "time" - initial time value, in milliseconds.
  • "lastTime" - last known time value, in milliseconds. Will be set to the value given in "timeInMs" before returning.
The object should be initialized using the idiom {} or new Object().
timeInMs ( Number )
A time value, in milliseconds. This could be the parameter received in a requestAnimationFrame() callback method. .
intervalInMs ( Number )
The length of the interval (animation cycle), in milliseconds.
Returns:
A value in the range [0, 1), where closer to 0 means "timeInMs" lies closer to the start, and closer to 1 means closer to the end of the interval. If an initial time wasn't set, returns 0. (Type: Number)
Example

The following code sets an angle of rotation, in degrees, such that an object rotated with the angle does a 360-degree turn in 5 seconds (5000 milliseconds). The variable time is assumed to be a time value in milliseconds, such as the parameter of a requestAnimationFrame() callback method. 

var angle = 360 * H3DU.getTimePosition(timer, time, 5000);

Finds the six clipping planes of a view frustum defined by a 4x4 matrix. These six planes together form the shape of a "chopped-off" pyramid (or frustum).

In this model, the eye, or camera, is placed at the top of the pyramid (before being chopped off), four planes are placed at the pyramid's sides, one plane (the far plane) forms its base, and a final plane (the near plane) is the pyramid's chopped off top.

Parameters:
matrix ( Array.<Number> )
A 4x4 matrix. This will usually be a projection-view matrix (projection matrix multiplied by view matrix).
Returns:
An array of six 4-element arrays representing the six clipping planes of the view frustum. In order, they are the left, right, top, bottom, near, and far clipping planes. All six planes will be unit vectors ("normalized" vectors with a length of 1). (Type: Array.<Array.<Number>>)
