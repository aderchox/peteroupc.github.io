Class: Camera

Camera

new Camera(scene, fov, near, far)

Source: extras/camera.js, line 228

A class for controlling the projection and view of a 3D scene, in the nature of an abstract "camera".

This class is considered a supplementary class to the Public Domain HTML 3D Library and is not considered part of that library.

To use this class, you must include the script "extras/camera.js"; the class is not included in the "glutil_min.js" file which makes up the HTML 3D Library. Example:

<script type="text/javascript" src="extras/camera.js"></script>
Parameters:
Name Type Description
scene glutil.Scene A 3D scene to associate with this camera object.
fov number Vertical field of view, in degrees. Should be less than 180 degrees. (The smaller this number, the bigger close objects appear to be.)
near number The distance from the camera to the near clipping plane. Objects closer than this distance won't be seen. This should be slightly greater than 0.
far number The distance from the camera to the far clipping plane. Objects beyond this distance will be too far to be seen.
