Class: Shape

Shape

new Shape(mesh)

Source: glutil.js, line 2177

An object that associates a geometric mesh (the shape of the object) with material data (which defines what is seen on the object's surface) and a transformation matrix (which defines the object's position and size). See the "Creating Shapes" tutorial.
Parameters:
mesh ( BufferedMesh )
A mesh in the form of a vertex buffer object. For glutil.Mesh objects, use the glutil.Scene3D#makeShape method instead.

setColor(r, g, b, aopt) → {glutil.Shape}

Source: glutil.js, line 2230

Sets material parameters that give the shape a certain color. (If a material is already defined, sets its ambient and diffusion colors.) However, if the mesh defines its own colors, those colors will take precedence over the color given in this method.
Parameters:
r ( Array.<number> | number | string )
A color vector or string, or the red color component (0-1).
g ( number )
Green color component (0-1). May be null or omitted if a string or array is given as the "r" parameter.
b ( number )
Blue color component (0-1). May be null or omitted if a string or array is given as the "r" parameter.
a ( number ) <optional>
Alpha color component (0-1). If the "r" parameter is given and this parameter is null or omitted, this value is treated as 1.0.
