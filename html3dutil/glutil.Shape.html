Class: Shape

Shape

new Shape(mesh)

An object that associates a geometric mesh (the shape of the object) with material data (which defines what is seen on the object's surface) and a transformation matrix (which defines the object's position and size). See the "shapes" tutorial.
Parameters:
Name Type Description
mesh BufferedMesh A mesh in the form of a vertex buffer object. For glutil.Mesh objects, use the glutil.Scene3D#makeShape method instead.

multQuaternion(quat) → {glutil.Shape}

Combines this shape's current rotation with another rotation described by a quaternion (a 4-element array for describing 3D rotations). The combined rotation will have the same effect as the new rotation followed by the existing rotation. Has no effect if a matrix was defined with glutil.Shape#setMatrix and the transform wasn't reset yet with glutil.Shape#resetTransform.
Parameters:
Name Type Description
quat Array.<number> A four-element array describing the rotation. A quaternion is returned from the methods glmath.GLMath.quatFromAxisAngle or glmath.GLMath.quatFromEuler.
Source:
  • glutil.js, line 2092
Returns:
This object. (Type: glutil.Shape )
Example
// Combine this shape's rotation with a rotation 20 degrees about the X-axis
shape.multQuaternion(GLMath.quatFromAxisAngle(20,1,0,0));
// Combine this shape's rotation with identity (no rotation)
shape.multQuaternion(GLMath.quatIdentity());
// Combine this shape's rotation with 30 degree pitch multiplied
// by 40 degree roll
shape.multQuaternion(GLMath.quatFromEuler(30,0,40));

multRotation(angle, v, vy, vz) → {glutil.Shape}

Combines this shape's current rotation with another rotation in the form of an angle and an axis of rotation. setRotation(angle, v, vy, vz) → {glutil.Shape}

Sets this object's rotation in the form of an angle and an axis of rotation. Has no effect if a matrix was defined with glutil.Shape#setMatrix and the transform wasn't reset yet with glutil.Shape#resetTransform.

If "v", "vy", and "vz" are omitted, this can instead be a 4-element array giving the axis of rotation as the first three elements, followed by the angle in degrees as the fourth element.
