Class: Shape

Shape

new Shape(mesh)

An object that associates a geometric mesh (the shape of the object) with material data (which defines what is seen on the object's surface) and a transformation matrix (which defines the object's position and size). See the "shapes" tutorial.
Parameters:
Name Type Description
mesh BufferedMesh A mesh in the form of a vertex buffer object. For glutil.Mesh objects, use the glutil.Scene3D#makeShape method instead.

setColor(r, g, b, a) → {glutil.Shape}

Sets material parameters that give the shape a certain color. However, if the mesh defines its own colors, those colors will take precedence over the color given in this method.
Parameters:
Name Type Description
r Array.<number> | number | string Array of three or four color components; or the red color component (0-1); or a string specifying an HTML or CSS color.
g number Green color component (0-1). May be null or omitted if a string or array is given as the "r" parameter.
b number Blue color component (0-1). May be null or omitted if a string or array is given as the "r" parameter.
a number Alpha color component (0-1). May be null or omitted if a string or array is given as the "r" parameter.
