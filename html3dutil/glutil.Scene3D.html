Class: Scene3D

Scene3D

new Scene3D(context)

Source: glutil.js, line 1429

A holder object representing a 3D scene. This object includes information on:
  • A projection matrix, for setting the camera projection.
  • A view matrix, for setting the camera's view and position.
  • Lighting parameters.
  • Shapes to be drawn to the screen.
  • A texture cache.
  • A screen-clearing background color.
When a Scene3D object is created, it compiles and loads a default shader program that enables lighting parameters, and sets the projection and view matrices to identity. The default lighting for the scene will have a default ambient color and one directional light source.
Parameters:
Name Type Description
context WebGLRenderingContext | object A WebGL 3D context to associate with this scene, or an HTML canvas element to create a WebGL context from, or an object, such as Scene3D, that implements a no-argument getContext method that returns a WebGL context.

setPerspective(fov, aspect, near, far) → {glutil.Scene3D}

Source: glutil.js, line 1711

Sets this scene's projection matrix to a perspective projection.

For considerations when choosing the "near" and "far" parameters, see glmath.GLMath.mat4perspective.

Parameters:
Name Type Description
fov number Y-axis field of view, in degrees. Should be less than 180 degrees. (The smaller this number, the bigger close objects appear to be.)
aspect number The ratio of width to height of the viewport, usually the scene's aspect ratio (getAspect() or getClientAspect()).
near number The distance from the camera to the near clipping plane. Objects closer than this distance won't be seen.
far number The distance from the camera to the far clipping plane. Objects beyond this distance will be too far to be seen.
Returns:
This object. (Type: glutil.Scene3D)
Example
// Set the perspective projection.  Camera has a 45-degree field of view
// and will see objects from 0.1 to 100 units away.
scene.setPerspective(45,scene.getClientAspect(),0.1,100);
