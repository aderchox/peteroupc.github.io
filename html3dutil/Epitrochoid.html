Class: Epitrochoid

Epitrochoid

new Epitrochoid(outerRadius, innerRadius, distFromInnerCenter)

Source: extras/evaluators.js, line 306

Parametric evaluator for a curve drawn by a circle that rolls along the outside of another circle, whose position is fixed, with a center of (0,0).

This class is considered a supplementary class to the Public Domain HTML 3D Library and is not considered part of that library.

To use this class, you must include the script "extras/evaluators.js"; the class is not included in the "h3du_min.js" file which makes up the HTML 3D Library. Example:

<script type="text/javascript" src="extras/evaluators.js"></script>
Parameters:
outerRadius ( Number )
Radius of the circle whose position is fixed.
innerRadius ( Number )
Radius of the rolling circle. An epicycloid results when outerRadius=innerRadius.
distFromInnerCenter ( Number )
Distance from the center of the rolling circle to the drawing pen. A prolate epitrochoid results when distFromInnerCenter is greater than innerRadius.
