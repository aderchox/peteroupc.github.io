Class: FrameBuffer

FrameBuffer

new FrameBuffer(context, width, height)

Represents an off-screen frame buffer.

When FrameBuffer's constructor is called, it will create a texture buffer with the given width and height and a depth buffer with the same dimensions, and will bind both to the frame buffer. The frame buffer currently bound to the WebGL context will remain unchanged.

Parameters:
Name Type Description
context WebGLRenderingContext | object WebGL context to associate with this buffer, or an object, such as Scene3D, that implements a no-argument getContext method that returns a WebGL context.
width number Width, in pixels, of the frame buffer. Fractional values are rounded up.
height number Height, in pixels, of the frame buffer. Fractional values are rounded up.
