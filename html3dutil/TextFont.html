Class: TextFont

TextFont

new TextFont()

Source: extras/text.js, line 158

Represents a bitmap font. NOTE: The constructor should not be called directly by applications. Use the TextFont.load method to get a TextFont object. This constructor's parameters are undocumented and are subject to change.

Methods

(static) load(fontFileName) → {Promise.<TextFont>}

Source: extras/text.js, line 641

Loads a bitmap font definition from a file. Note that this method only loads the font data and not the bitmaps used to represent the font.
Parameters:
fontFileName ( string )
The URL of the font data file to load. The following file extensions are read as the following formats:
  • ".xml": XML
  • ".json": JSON
  • ".bin": Binary
  • ".fnt": Text or binary
  • All others: Text
Returns:
A promise that is resolved when the font data is loaded successfully (the result will be a TextFont object), and is rejected when an error occurs. (Type: Promise.<TextFont>)

measure(str, height) → {Array.<number>}

Source: extras/text.js, line 205

Calculates the width and height of a text string when drawn using this font.
Parameters:
str ( String )
A text string.
height ( Number )
The line height to use when measuring the text. Cannot be less than 0.
Returns:
An array of two numbers; the first is the width of the string, and the second is the height of the string (taking into account line feed characters, U+000A, that break lines). (Type: Array.<number>)
